RoleScope

ERDColumn
PluTx :: PluTxDataModel :: Data Model :: Function :: RoleScope

Description

Role scope indicates whether a certain function is can be added to/removed from the accessibility list of a role by any users who have the right to manage the system roles.

Role Scope value must be one of the followings:

Global BODs System Wallet

Global scope functions will be granted to the Global Role-typed role at the deployed time. These functions are mandatory and static and immutable from system perspective.

BODs scope functions will be granted to the BODs Role-typed role at the deploy time. These functions are mandatory and static and immutable from system perspective.

System scope functions can be granted to/ungranted from other system roles that is not of type global and wallet.

Wallet scope functions are granted to wallet roles only at the deployed time.
