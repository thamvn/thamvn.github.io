Status

ERDColumn
PluTx :: PluTxDataModel :: Data Model :: Rule :: Status

Description

Status must be one of the following values:

New Activate Deactivate

  • New: Newly created, no signature required
  • Activate: launched to be used. Signature required
  • Deactivate: stop using. Signature required. Cannot activate again.
