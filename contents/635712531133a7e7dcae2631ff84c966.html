Status

ERDColumn
PluTx :: PluTxDataModel :: Data Model :: User :: Status

Description

User status must be one of the following value:

New Active Locked Deactive

  • New: newly created user account
  • Active: user has logged in this account and changed their password
  • Locked: user input wrong password for a number allow times (say e.g 5 times)
  • Deactive: this user account has been deactivated
